Solar System Simulator (S.S.S.)

About Us

What is S.S.S.?

S.S.S. is a product that was created for the EECS 448 class at KU. It is a semi-accurate solar system simulator where the user can add and customize new or existing planets. Furthermore, the simulator plays music as well through a custom-made synthesizer.

How do I use S.S.S.?

S.S.S. was created to be simple, to move the camera simply right click and drag. You can increase the speed at which the planets move by moving the slider at the bottom of the screen. To add randomly generated planets click the 448 planets button. To view the solar system with full lighting you can turn on the lights using the button in the top right, if you want to see how the sun would light up the planets, you can turn them off using the same button.
